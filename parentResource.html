Parent Resources

Does your child like to solve problems? They might be interested in coding, and it's never too early or too late to start! The computer science industry is rapidly growing, and learning to code at a young age gives your child an edge in an increasingly competitive workforce. Coding also fosters incredible creativity and builds up critical thinking ability, both important life skills to have. We know parents always want what's best for their kids, so here are some resources to start your child off on their coding adventure. These are all great sites designed to make learning a blast instead of a bore!

Code.org

Code.org is a nonprofit dedicated to bringing computer science to all schools, and has done so with their Hour Of Code program campaign that has worked with 15% of all students globally. The site offers free computer science courses and allows kids to create fun coding projects, with material for all ages, even young kids 4-6.

Visit this site!
Code Combat

This site, for ages 5-17, allows users to learn Python or Javacript through an RPG-roleplaying game. This fun and innovative take on coding guides kids through different worlds as they progress, making learning programming fun!

Visit this site!
Code Wars

Bringing a competitive game aspect into coding, this free site uses the Japanese martial arts practice of kata as an incentive to learn how to code in the language of your choice-they have a lot to choose from!. Geared towards older kids, you have the ability to look at other users' code, allowing a deeper understanding of the language and process.

Visit this site!
